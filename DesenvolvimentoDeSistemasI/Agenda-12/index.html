Youtube

YouTube é a maior plataforma de compartilhamento de vídeos do mundo, permitindo que usuários façam upload, assistam e compartilhem vídeos em uma ampla variedade de categorias, incluindo música, entretenimento, educação e muito mais.

Link: Youtube


Google

Google é o mecanismo de busca mais utilizado globalmente, oferecendo resultados relevantes para pesquisas na web em milissegundos. Além do buscador, Google fornece uma gama de serviços, como Google Maps, Google Drive, Gmail e Google Photos, todos integrados para facilitar a experiência online dos usuários.

Link: Google


WhatsappWeb

WhatsApp Web é a versão para desktop do popular aplicativo de mensagens, permitindo que os usuários enviem mensagens, compartilhem arquivos e façam chamadas diretamente do navegador. Sincronizando com o aplicativo móvel, oferece conveniência para se comunicar enquanto trabalha ou navega na internet.

Link: WhatsappWeb


Outlook

Outlook é um serviço de gerenciamento de e-mails e calendário desenvolvido pela Microsoft. Oferecendo uma interface intuitiva, recursos de organização e integração com outras ferramentas do Microsoft 365, permite que os usuários gerenciem suas comunicações, compromissos e tarefas de forma eficiente.

Link: Outlook


GitHub

GitHub é uma plataforma de hospedagem de código-fonte e controle de versão, amplamente utilizada por desenvolvedores para colaborar em projetos de software. Oferecendo ferramentas para revisão de código, gerenciamento de projetos e documentação, é um hub essencial para a comunidade de desenvolvimento open-source e empresas.

Link: GitHub
