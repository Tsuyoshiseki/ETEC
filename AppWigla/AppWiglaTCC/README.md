Instruções de Instalação

1. Instalar Dependências

⦁	.NET 8 SDK
⦁	Visual Studio 2022 com workload ".NET MAUI"
⦁	XAMPP (MySQL + Apache)


2. Importar Banco de Dados

Abra XAMPP e ligue:

⦁	Apache
⦁	MySQL

1.	Abra phpMyAdmin
2.	Clique em Importar
3.	Escolha o arquivo wigla_db.sql
4.	Clique em Executar

O banco será criado automaticamente.


3. Rodar o projeto

No Visual Studio:

Selecione Windows Machine
